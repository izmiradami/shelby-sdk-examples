# Shelby SDK Examples

A simple example project demonstrating how to use the Shelby Protocol SDK with Aptos.

## Features

- Upload blobs
- Download blobs
- Shelby Testnet
- Aptos TypeScript SDK

## Installation

```bash
npm install
```

## Upload

```bash
node upload-test.mjs
```

## Download

```bash
node download-test.mjs
```

## Requirements

- Node.js 22+
- Shelby API Key
- ShelbyUSD balance
